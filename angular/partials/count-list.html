Introduction

This site presents detailed results estimated for the Moreland North-east Ward 2016 election.

The count and code used to perform it are based on the Austrlian Senate count method as implemented by dividebatur

Some very big assumptions are made here:

  1. Only first preference counts are currently available
  2. An estimated flow of preferences is made using the registered how-to-vote card of each candidate
    • This is a very crude assumption. Particularly in a progressive council like Moreland, many voters will not follow how-to-vote cards.
    • How-to-vote cards for two candidates were not available. In these cases an preferences are assumed to flow in same order as a select aligned candidate. This may be a very poor assumption indeed.
    • Big thanks to John Englart whose excellent guide includes the compilation of the how-to-vote card information.

A big shout-out to Graham Bowland for the original dividebatur to count the Australian Senate results. As with the original dividebatur code, this is completely open source software distributed under the Apache License, 2.0.
